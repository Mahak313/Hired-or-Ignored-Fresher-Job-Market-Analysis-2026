## 📌 Project Overview
An end-to-end Data Analytics project that analyzes 
**why freshers are still not getting jobs in 2026.**

Built using real survey data from 50 freshers + 
90 Data Analyst job listings.

---

## 🎯 Problem Statement
> "Freshers are applying everywhere but getting no response. 
> Is it a skills problem, a resume problem, or a market problem?"

## 📊 Key Insights

### 1️⃣ 38% Freshers Got ZERO Interviews
> Despite having SQL, Python and Power BI skills —
> 38% freshers received zero interview calls

### 2️⃣ Resume Review = 2x Interview Chances
> Freshers who got resume reviewed — 78% got interviews
> Freshers who did NOT — only 43% got interviews

### 3️⃣ Skills Gap — Tableau is Most Ignored!
> Market demands Tableau in 54 jobs
> Only 28 freshers have it — 26 skill gap!

### 4️⃣ Quality Over Quantity
> 5-10 targeted applications = better results
> than 20+ random applications daily

### 5️⃣ Only 23% Jobs Are Fresher Level
> Out of 90 Data Analyst jobs —
> only 21 are fresher level positions
